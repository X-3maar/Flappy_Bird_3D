# Flappy_Bird_3D
This game is a new 3d version of flappy bird with some more cool features.
I love remaknig old games so that's why i made this game with better graphics.
<img width="1366" height="768" alt="image" src="https://github.com/user-attachments/assets/e942164f-061e-4815-8a8b-df25b1adacdb" />
## Controls
In the real game, there is only one simple control button to play which is space/left click. but in this version, i added a toggle camera button that allows you to switch between first person and normal view, try pressing "c" to test this feature.
<img width="1105" height="568" alt="image" src="https://github.com/user-attachments/assets/6ef8f9e0-28f5-42a6-bd8f-ed42b335df49" />
Those are a few more control that you may need to know: 
Space/click to jump, esc to pause, space to restart.

### Further information
this game was made using godot game engine, all the assets are manually designed by me using CSG (Constructive Solid Geometry) and mesh instances (Except for the 2d assets and fonts).
